Belajar CSS untuk mendesain Web

Pengenalan Dasar CSS

Pemahaman CSS sangat penting bagi front-end web developer.


CSS digunakan untuk mendesain situs web, dengan CSS element HTML akan dapat didesain sedemikian rupa sehingga tampilan situs web menjadi lebih cantik.


Pada artikel kali ini kita akan membahas lebih lanjut mengenai CSS, dimulai dari pengertian, cara penulisan, syntax, istilah-istilah CSS, dan CSS box model.

Yuk cari tau...

Apa itu CSS?

CSS (Cascading Style Sheet) merupakan sebuah bahasa pemrograman yang digunakan untuk mengatur atau mendesain tampilan web sehingga terlihat lebih menarik dan indah.


CSS menjelaskan bagaimana elemen HTML ditampilkan, seorang web developer dapat mengatur layout, warna, font, garis background, animasi, dan lain-lain pada halaman webnya.








Selanjutnya kita cari tau tentang cara penulisan CSS

Bagaimana Cara Menulis CSS di HTML?

Untuk menulis CSS di HTML ada tiga cara yaitu:



  1. Internal CSS --> ditulis di dalam tag <style>
  2. Inline CSS --> ditulis di atribut elemen HTML
  3. External CSS --> ditulis di file CSS terpisah dengan HTML


Berikut penjelasannya...


1. Internal CSS

Cara penulisan kode CSS secara internal yaitu kode CSS ditulis di dalam tag <style>. Metode penulisan CSS secara Internal juga dikenal dengan sebutan Embeded CSS.

Tag <style> bisa ditulis di dalam tag <head>, bisa juga ditulis di dalam tag <body>. Namun umumnya ditulis di dalam <head>.


2. Inline CSS

Cara penulisan kode CSS secara inline yaitu kode CSS ditulis langsung pada atribut <style> di elemen HTML.

Kode CSS ditulis langsung tanpa menggunakan kurung kurawal {...}. Namun penulisan properti tetap dipisah dengan tanda titik koma.

Inline CSS biasanya digunakan untuk memberikan style pada satu elemen saja. CSS pada inline akan lebih diprioritaskan dibandingkan dengan internal dan eksternal.


3. External CSS

Eksternal CSS adalah kode CSS yang ditulis terpisah dengan kode HTML. Eksternal CSS ditulis di dalam file khusus yang berekstensi .css.

Untuk menggunakan metode CSS External, file CSS harus dihubungkan dengan file HTML. Ada 2 cara yang dapat dilakukan untuk menghubungkan file CSS ke HTML yaitu dengan menggunakan tag <link> dan menggunakan @import.

Istilah-Istilah CSS


  • Declaration
  • Selector
  • Property
  • Value




1. Declaration

Declaration merupakan satu baris code pada CSS (tidak termasuk comment) yang biasanya diapit oleh tanda kurung kurawal {...}

Semua sintaks setelah kurung kurawal pertama dan sebelum kurung kurawal terakhir (dan tidak termasuk komentar) adalah sebuah declaration. Berikut contoh dari declaration. Untuk dua atau lebih declaration, dapat ditulis seperti pada contoh dibawah.

2. Selector

Pada CSS, selector digunakan untuk memilih elemen mana dari HTML yang akan ditargetkan atau dipengaruhi bersama dengan property dan value. Seperti contoh, apabila kita ingin mengatur posisi teks atau warna teks, cara memilihnya adalah dengan memakai selector.

Berikut contohnya, kode di bawah yang menjadi selector yaitu tag <p>. Selector akan membuat seluruh elemen yang berada di tag <p> terpengaruh oleh property dan value yang diberikan, yaitu posisi teks-nya akan berada di tengah dan teks akan berwarna merah.

3. Property

Property pada CSS digunakan untuk menambahkan atau mengatur ukuran teks, jenis font, warna teks, warna background, dan sebagainya.

Property muncul sebelum titik dua (:) dan sesudah titik koma (;) di setiap barisnya, seperti pada gambar di bawah ini. Pada gambar di bawah font-size, font-weight, border, dan color yang merupakan bagian property.

4. Value

Value adalah kode CSS yang mendefinisikan pengaturan untuk property CSS. value atau nilai ditulis setelah titik dua (:) di setiap barisnya.

Pada contoh di bawah, kode tersebut memiliki selector h1 yang diberikan property font-size dengan value'24px'.

CSS Box Model


CSS Box Model adalah sebuah konsep dimana setiap element yang terdapat pada halaman web diproses sebagai kotak (box). Mulai dari paragraf, header, form, gambar, logo hingga video, sebenarnya di tampilkan oleh web browser sebagai ‘box’.

Sebagaimana layaknya ‘kotak’, masing-masing element HTML ini terdiri dari 4 lapisan, yakni: konten (isi), padding, border dan margin. Keempat ‘lapisan’ inilah yang membangun CSS Box Model.







Bagian CSS Box Modeling Deskripsi
Content Content merupakan isi dari elemen yang letaknya di tengah-tengah pada box model. Biasanya berupa teks atau gambar.
Padding Padding digunakan untuk membuat ruang di sekitar konten elemen, di dalam batas yang ditentukan (padding, padding-top, padding-right, padding-bottom, padding-left).
Border Border merupakan garis tepi pembatas element. Kita bisa mengatur berbagai hal tentang border, seperti ketebalan, warna, dan jenis garis yang digunakan.
Margin Margin berguna untuk mengatur ruang (space) di luar elemen. (Margin, margin-top, margin-right, margin-bottom, margin-left).
